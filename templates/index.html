Brain Tumor Detection Web Application

The Brain Tumor Detection Web Application is a web-based tool that leverages object detection algorithms like Faster R-CNN to analyze MRI images and automatically detect the presence of brain tumors. This application is designed to assist doctors and radiologists in diagnosing tumors quickly and accurately using a custom dataset.
